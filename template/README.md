# Шаблон промпта для генерации персонализированных email-рассылок
## Описание шаблона
Предназначен для автоматизации создания приветственных писем новым пользователям в CRM-системе.
Параметры:

`{user_name}` – имя пользователя (обязательный)

`{product_feature}` – ключевая функция продукта, релевантная интересам пользователя (обязательный)

`{discount_code}` – персональный промокод (опциональный)

## Особенности реализации
* Умная параметризация
* Условный блок {% if discount_code %} автоматически адаптирует инструкции под наличие промокода
* Динамические требования к CTA исключают ошибки при отсутствии опционального параметра
* Anti-pattern protection
* Запрет на шаблонные фразы ("рады приветствовать") повышает уникальность контента
* Жёсткие ограничения на длину (максимум 4 предложения) предотвращают "размытые" ответы
